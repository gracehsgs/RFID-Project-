The RFID Attendance System is a digital system designed to replace time-consuming and error-prone manual attendance systems. Attendance data is recorded automatically and accurately with a time stamp when an RFID card is scanned.

The system uses an ESP32 as the main processor, which connects to various modules, including an RFID reader, an OLED screen, a real-time clock (RTC) for accurate timekeeping and a MicroSD card for local storage. This data is also sent in real time to an MQTT broker and can be forwarded to Google Sheets using Node-RED.
